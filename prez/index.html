React : l’outil

Objectifs

  • Virtual DOM (approche inverse de jquery, DOM généré optimisé en accès grâce aux data-reactid)
  • Data flow libre
  • permet l’isomorphisme => SEO, accessibilité, performance
  • rendu (vue) uniquement, indépendant du contexte d’utilisation

Concepts clés

  • Composants (création par classe)
  • Méthode de rendu
  • Communication hirerarchique entre composants (props)
  • Etat d’un composant (state)
  • Event listener

Orienter son modèle de données pour les performances

Pour obtenir les meilleurs performances possibles : nécessite l’utilisation d’un modèle de donnée immuables par forcément facile à mettre en oeuvre. Sans modèle immuable, plus difficile d’optimiser les perfs. Pas forcément obligatoire.
